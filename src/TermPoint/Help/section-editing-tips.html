Section Editing Tips

Two features in the section editor can save a lot of repetitive clicking once you know they're there: smart section-code incrementing when you copy, and block-pattern propagation when you set meeting times.

Copying a Section Increments the Code Automatically

When you copy a section, TermPoint doesn't just duplicate it — it also advances the section code. The rule is simple: if the code ends in a number, that number goes up by one. If the next number is already taken, you'll see a notice and the code field is left blank for you to fill in manually.

Original code Copy gets
AB2 blank (no number to increment)
01 02
HY1 HY2
ONLINE3 ONLINE4

The copied section opens immediately in the editor with all other fields carried over from the original. Adjust whatever needs to change — room, instructor, meeting time — and you're done.

This pairs well with block patterns (see below). Copy a section, pick a different time slot, and you have a second offering of the same course in seconds.

Setting Meeting Times with Block Patterns

Most courses follow one of a handful of standard weekly rhythms — MWF, TR, a single long block on Monday, and so on. Block patterns let you express that rhythm once and have TermPoint fill in all the meeting days for you.

What is a block pattern?

A block pattern is a named set of weekdays — for example, MWF (Monday, Wednesday, Friday) or TR (Tuesday, Thursday). You define them once in Settings → Block Patterns and they're available everywhere in the section editor from then on.

Block patterns are set up per academic year under Settings → Block Patterns in the top menu bar. If your institution's standard patterns aren't already there, add them once and they'll be available for every section in that year.

How propagation works

In the section editor, select a Block Pattern from the dropdown, then choose a Block Length (e.g. 50 min, 75 min, 3 hours) and the Start Time for the first meeting in the pattern.

TermPoint immediately creates all the remaining meetings — same length, same start time — one for each day in the pattern. You don't have to touch them individually.

Need one day to differ? Every individual meeting is editable. Change the start time or duration on any single day after propagation and the others are unaffected. Propagation is a one-time fill, not a permanent link.

A typical workflow

  1. Open or create a section and unlock the meeting fields.
  2. Select Block Pattern — e.g. MWF.
  3. Choose Block Length — e.g. 50 min.
  4. Choose Start Time — e.g. 10:00 AM.
  5. TermPoint fills in Monday 10:00–10:50, Wednesday 10:00–10:50, Friday 10:00–10:50.
  6. Edit any individual day if needed. Done.
