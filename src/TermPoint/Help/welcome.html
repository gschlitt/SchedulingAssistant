Welcome to TermPoint

TermPoint is a scheduling visualization and management tool for university administrators. It lets you see at a glance how course sections fit together across the week, and gives you the tools to add, edit, and organize them.

Use the topic list on the left to jump to any help section. If you're new here, start with Getting Started.

What You Can Do

📅 Visualize Schedules

See all sections laid out on a weekly grid, hour by hour, day by day.

✏️ Manage Sections

Add, edit, copy, and organize course sections with an inline editor.

👩‍🏫 Track Instructors

Assign instructors, track workload, and record standing commitments.

🔍 Filter & Focus

Narrow the grid to a specific instructor, room, department, or tag.

📤 Export

Export the schedule as a PNG image or generate workload reports.

🗓️ Multiple Semesters

Compare Fall and Spring (or any combination) side by side.

About This Help System

The help topics on the left are plain HTML files stored in the Help/ folder next to the application. You can edit them in any text editor to add your own notes, screenshots, or links. Each page can include one or more YouTube video embeds — see the commented-out <iframe> blocks in each file for the pattern to follow.
