Creating and editing a section

  1. Specify a subject and course number
  2. Pick a section code format (These are defined under Configuration and greatly speed up editing)
  3. The section code is automatically supplied (but can be overridden if necessary)
  4. Optionally, pick a section type
  5. Either add meetings with +Meeting, or choose a block pattern (defined under Configuration)
  6. Set scheduling details for the meetings. The available block lengths are determined by the start times (as defined under Configuration). You can override if necessary.
  7. Optionally, choose an instructor and associated workload. Workload View will be automatically updated
  8. Optionally, choose tags, reserve codes and resources (all configurable under Scheduling Environment)
  9. Click 'Apply' to register the edits. Don't forget to periodically save your edits (or use Autosave).
