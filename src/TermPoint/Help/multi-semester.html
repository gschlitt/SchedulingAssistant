Multi-Semester Editing

TermPoint can display and edit sections from more than one semester at the same time, letting you compare Fall and Spring (or any other combination) on a single grid.

This article is a placeholder. Edit Help/multi-semester.html to add full content.

Enabling Multiple Semesters

Use the semester picker in the top toolbar to add additional semesters to the current view. Each semester is shown in a distinct color so you can tell them apart at a glance.

Editing in Multi-Semester Mode

The Sections list on the left shows sections for all active semesters. Use the semester indicator on each card to identify which semester a section belongs to before editing.
