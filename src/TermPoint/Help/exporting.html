Exporting

TermPoint can export your schedule data in several formats. All export options are available under the Export menu in the toolbar.

Export Schedule PNG

Captures the current Schedule Grid exactly as it appears — including any active filters and multi-semester colouring — and saves it as a PNG image.

  • Go to Export → Export Schedule PNG…
  • Choose a file location and name.
  • The exported image uses a dark-on-light colour scheme optimised for printing.

Set your filters before exporting to get a focused view, such as a single instructor's schedule or a specific set of rooms.

Workload Report

Produces a plain-text summary of each instructor's assigned sections across the selected semesters, grouped by instructor.

  • Go to Export → Workload Report…
  • Select the academic year and semesters to include.
  • Copy the report text or save it to a file.

Workload Mailer

Generates a personalised email for each instructor containing their scheduled sections. TermPoint opens each message in your default email client so you can review and send it individually.

Setup

  1. Go to Export → Workload Mailer…
  2. Select the academic year and the semesters to include.
  3. Choose which instructors to email (those without an email address on file are greyed out).
  4. Edit the subject and body templates. Use the placeholder legend shown in the panel:
    • {FirstName}, {LastName} — instructor's name
    • {AcademicYear} — e.g. "2025–2026"
    • {Semester} — e.g. "Fall"
    • {Workload} — the generated workload block for this instructor
  5. Click Start Sending. Each instructor's rendered email opens in your mail client. Review it, click Send in your email app, then return to TermPoint and click Next to continue to the next instructor.

TermPoint uses mailto: links — it does not send email directly. Your default email client (Outlook, Apple Mail, etc.) handles the actual sending.
