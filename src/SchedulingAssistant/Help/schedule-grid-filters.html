Filtering the Schedule Grid

The filter panel lets you narrow the grid to show only the sections you care about. Open it by clicking the filter icon (funnel) in the top-right corner of the Schedule Grid panel.

Filter Dimensions

You can filter by any combination of these eight dimensions:

  • Instructors — show only sections taught by selected instructors
  • Rooms — show only sections meeting in selected rooms
  • Subjects — e.g. HIST, MATH, ENGL
  • Campuses — if your institution uses multiple campus codes
  • Section Types — e.g. Lecture, Lab, Seminar
  • Tags — arbitrary labels you can attach to sections
  • Meeting Types — e.g. In-Person, Online, Hybrid
  • Levels — e.g. 100-level, 200-level, Graduate

AND / OR Logic

The filter uses two-level logic:

  • Selections within a dimension are combined with OR (a section matches if it belongs to any of the chosen values).
  • Selections across dimensions are combined with AND (a section must satisfy all active filters simultaneously).

Example: Checking Prof. Smith and Prof. Jones under Instructors, plus Room 201 under Rooms, will show sections taught by either Smith or Jones that also meet in Room 201.

The Filter Badge

When any filter is active, an amber badge appears on the filter icon showing how many dimensions are currently filtering. This reminds you that the grid is not showing everything. Click the icon again and choose Clear All to reset.

Overlays vs. Filters

Overlays are different from filters. An overlay highlights matching sections in red while keeping all other sections visible (greyed out). Access overlays by right-clicking a tile or from the filter panel's Overlay tab.
