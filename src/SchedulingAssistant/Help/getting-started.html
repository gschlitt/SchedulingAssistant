Getting Started

This page walks you through the first steps you need to take to get set up.

Getting your data into TermPoint

The start-up wizard guided you through some basic configuration (section codes, block structure and so on). It's time to enter subjects, courses and instructors

Courses

  • In the top menu, select 'Courses' and use the bottom dialog to enter the Subjects you schedule, along with their calendar codes. For example a Biology department might schedule courses in both 'Biology' ('BIOL') and 'BioChemistry' (''BIOC').
  • With subjects in place, you can begin entering your courses. Note that using Tags to filter your views can be very effective. If you can already think of some Tags that might be useful (perhaps 'PreMed' or 'Honours') create them first (under 'Scheduling Environment'). If you then choose some of those tags for the courses you create, every section of those courses will automatically inherit those tags. But you can always do this later if you wish.

With courses in place, you can already begin scheduling: select an academic year and semester (top menu) and enter sections in the 'Section View' pane on the left. (There's more help on this available if you need it.) But you might want to enter some instructors first.

Instructors

  • In the top menu, select 'Instructors' and use the dialog to enter instructor details. Selecting an instructor will allow you to enter any relevant scheduling information for the semesters you're about to schedule.

Scheduling Environment

Under 'Scheduling Environment' you can enter the various aspects sections might have (a 'Type' for example, (perhaps 'Online')). If you're responsible for rooming your sections, enter room types and rooms here. Tags provide a particularly effective means of filtering your semester view to just the information you need. An example tag might be 'Cohort 1': which you could use to tag the sections serving a certain student cohort. Then filtering by that tag eliminates all the clutter, allowing you to see just the sections involved.

Configuration

The start-up wizard guided you through setting these general scheduling parameters. But you can always revise them if you wish, under 'Configuration.'
