The Schedule Grid

The Schedule Grid is the primary view in TermPoint. It displays all sections for the selected semester(s) as colour-coded tiles on a Monday–Friday, 08:30–22:00 canvas.

Anatomy of a Tile

Each tile represents one section meeting. Information is packed onto a single line to conserve vertical space: 

HIST 101 A  JRS
  • HIST 101 — subject and course number
  • A — section code
  • JRS — instructor initials

Co-scheduled sections

When two or more sections share the exact same start time and duration they appear stacked inside one tile, separated by a thin rule. This is common for cross-listed courses.

Overlapping sections

Sections that overlap in time but are not co-scheduled appear side by side in the same day column, each at its correct vertical position.

Selecting a Section

Click any tile to select the section. The selection is synchronised across all three panels: the section is highlighted in the Section List on the left, and the Workload Panel scrolls to the relevant instructor row.

Double-click a tile to jump directly to that section's editor in the Section List panel.

Right-click Context Menu

Right-clicking a tile shows a context menu with quick actions:

  • Edit — opens the inline editor
  • Copy — creates a duplicate of the section below it in the list
  • View Instructor Overlay — highlights all sections for this instructor in red, including any standing commitments

Multi-Semester View

When more than one semester is checked in the top-right Semester selector, the grid shows sections from all selected semesters simultaneously. Each semester is assigned a distinct background colour so you can tell them apart at a glance. A small semester-name badge appears in the bottom-right corner of each tile.

Sections that exist in multiple semesters carry a small indicator dot in the Section List to show how many semesters they appear in.
